# I425 Course Project

Shared starter repository for the INFO-I 425 course project.

## Project Purpose

This repository holds the team's Restaurant Ordering System course project files, API code, documentation, and collaboration updates.

## Lab 1 Files

- `docs/Restaurant_Ordering_System_Proposal.docx`: project proposal.
- `database/restaurant_ordering_system_db.sql`: MySQL database schema and sample data.
- `docs/api-endpoint-documentation.md`: RESTful API endpoint documentation.
- `docs/Lab1_Requirements_Fulfillment.docx`: explanation of how the Lab 1 requirements are fulfilled.
- `docs/lab1-requirements-fulfillment.md`: Markdown copy of the requirements summary.

## Team Workflow

1. Clone the repository.
2. Create or edit project files locally.
3. Commit changes with clear messages.
4. Push updates to GitHub so the team can review the shared project.
